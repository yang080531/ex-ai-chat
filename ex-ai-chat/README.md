## 🚀 使用方法

### 1. 配置API（首次使用）
1. 点击"API配置"
2. 选择AI服务商（推荐Kimi/豆包，国内访问快）
3. 输入API Key（从对应官网获取）
4. 选择模型，调整温度参数
5. 保存配置

### 2. 设定角色
1. 点击"角色设定"
2. 上传TA的照片（可选）
3. 导入聊天记录txt文件（可选，自动分析风格）
4. 填写性格、说话风格、口头禅
5. 添加共同回忆和分手原因
6. 点击"生成人设"

### 3. 开始对话
1. 点击"开始对话"
2. 输入消息，像正常聊天一样
3. AI会以前任的身份和语气回复

## 📦 打包成APK

### 方法一：Capacitor（推荐）

#### 1. 准备工作
确保已安装：
- Node.js 18+ (https://nodejs.org/)
- Android Studio (https://developer.android.com/studio)

#### 2. 安装依赖
```bash
cd ex-ai-chat
npm install
```

#### 3. 添加安卓平台
```bash
npx cap add android
```

#### 4. 同步并打开
```bash
npx cap sync
npx cap open android
```

#### 5. 生成APK
在Android Studio中：
1. 等待Gradle同步完成
2. 点击菜单 `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
3. APK位置：`android/app/build/outputs/apk/debug/app-debug.apk`

### 方法二：GitHub Actions自动打包

1. 把代码上传到GitHub仓库
2. 已配置`.github/workflows/build-apk.yml`
3. 每次推送代码自动构建APK
4. 在Actions页面下载APK

### 方法三：在线打包工具

如果不想装环境，可以使用：
- [WebIntoApp](https://www.webintoapp.com) - 上传ZIP打包
- [AppMaker](https://appmaker.merq.org) - 德国免费服务

## 🐛 常见问题

**Q: 为什么回复很慢？**  
A: 取决于API服务商和网络。推荐用Kimi或豆包，国内访问快。

**Q: 可以离线使用吗？**  
A: 不行，需要连接大模型API。

**Q: API调用收费吗？**  
A: 是的，各服务商都有免费额度，用完需要充值。一般一次对话0.01-0.1元。

**Q: 人设设定不生效？**  
A: 确保点击了"生成人设"按钮，看到生成的Prompt才算保存成功。

**Q: 聊天记录怎么清空？**  
A: 聊天页面点击右上角垃圾桶图标，或在手机设置中清除应用数据。
